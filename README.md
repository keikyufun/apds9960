# APDS-9960（micro:bit 用拡張）

APDS‑9960 の RGB・Clear・近接・ジェスチャーを micro:bit で扱うための拡張機能です。  
初期化ブロックを実行するだけで、色・近接・ジェスチャーが取得できます。

---

## 使い方（MakeCode）

MakeCode の「拡張機能」→「URL から追加」で、次の URL を入力してください。

https://github.com/keikyufun/APDS-9960#v0

読み込むと「APDS9960」というカテゴリが追加されます。

---

## 追加されるブロック

- APDS9960 初期化  
- Clear値  
- 赤  
- 緑  
- 青  
- 近接値  
- ジェスチャー（up / down / left / right / none）  
- 色を判定（red / green / blue / white / black / unknown）
